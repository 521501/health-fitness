Health & Fitness App

This project was developed as a comprehensive solution for personal health management, aiming to bridge the gap between logging daily physical activities and visualizing long-term health trends. By utilizing the MERN stack (MongoDB, Express, React, and Node.js), I built this application to handle data-intensive tasks like tracking calories, workout durations, and daily steps, providing users with a clean and actionable dashboard for their fitness journey.

Core Features
Personalized Dashboard: A centralized hub designed to give users a quick snapshot of their daily health metrics and progress toward their fitness goals.

Workout & Activity Tracking: Allows for seamless logging of various physical activities, ensuring users can accurately record their effort throughout the day.

Data Visualization: Incorporates interactive charts to help users identify patterns in their exercise routine and weight-tracking journey over time.

Secure User Management: Implements authentication flows to ensure that personal health and fitness data remains private and secure.

Technical Architecture
This application is built on the MERN stack, chosen for its efficiency in handling full-stack JavaScript development:

Frontend: Built with React.js to create a responsive and dynamic user interface that functions smoothly across different devices.

Backend: Powered by Node.js and Express.js to manage server-side logic and API requests efficiently.

Database: Uses MongoDB for flexible, scalable storage of user profiles and activity logs.

Getting Started
To get a local copy of this project running on your machine, follow these simple steps:

Prerequisites
Ensure you have Node.js installed globally on your system.

You will need a MongoDB account (or a local instance) to manage the database connection.

### Installation
1. Clone the repository:
   git clone <https://github.com/521501/health-fitness>
2. Setup the Backend:
   npm install
3. Setup the Frontend:
   cd client
   npm install
4. Configure Environment Variables:
   Create a .env file in the root folder and add your MONGO_URI.
5. Run the Application:
   npm run dev
